Q-VLM 项目结构分析文档

项目名称：Q-VLM (Quantization for Large Vision-Language Models)

发表会议：NeurIPS'24

核心特性：W4A4 (4-bit 权重 + 4-bit 激活) 量化方法

1. 项目概述

Q-VLM 是一个用于大型视觉-语言模型后训练量化的框架，实现了高效的 W4A4 量化方法。

核心特性：

  • W4A4 量化：权重和激活都使用 4-bit 量化
  • 非对称量化：使用 asymmetric_linear_quantization_params() 计算 scale 和 zero-point
  • 动态激活范围搜索：基于熵的搜索策略 (DED - Distribution Entropy Difference)
论文链接：https://arxiv.org/abs/2410.08119

4. 实验脚本说明

4.1 generate_sqa_response.sh

生成量化模型的预测答案 (带校准流程):

CUDA_VISIBLE_DEVICES=0 python -m llava.eval.model_vqa_science \
    --model-path  \
    --question-file //ScienceQA/data/scienceqa/llava_test_QCM-LEPA.json \
    --image-folder //ScienceQA/data/scienceqa/images/test \
    --question-file-calibrate //ScienceQA/data/scienceqa/llava_train_QCM-LEPA.json \
    --image-folder-calibrate //ScienceQA/data/scienceqa/images/train \
    --answers-file //LLaVA/results/ScienceQA/LLaVA-vicuna-7B-v1.3-4bit.jsonl \
    --conv-mode llava_v1 \
    --load-4bit

参数说明：

  • --model-path：预训练 LLaVA 模型路径
  • --question-file：测试集问题文件
  • --question-file-calibrate：校准集 - 用于确定量化范围
  • --load-4bit：启用 4-bit 量化

4.2 evaluate_sqa_response.sh

评估模型生成的答案并计算准确率:

python ../llava/eval/eval_science_qa.py \
    --base-dir //ScienceQA/data/scienceqa \
    --result-file //LLaVA/results/ScienceQA/LLaVA-vicuna-7B-v1.3-4bit.jsonl \
    --output-file //LLaVA/results/ScienceQA/test_llava-7b_output.json \
    --output-result //LLaVA/results/ScienceQA/test_llava-7b_result.json

def compute_DED(self, p_k, p_k1):
    """Distribution Entropy Difference - 核心创新"""
    return -1 * torch.sum(joint_p * torch.log(condition_p), dim=1).mean()

def cal_entropy(self, attn):
    """熵计算"""
    return -1 * torch.sum((attn * torch.log(attn+1e-7)), dim=1).mean()

6. 工作流程图

6.1 量化推理流程

  1. generate_sqa_response.sh
    • 加载 4bit 量化模型
    • 校准阶段 ← 使用训练集校准激活范围
    • 推理测试集 → 预测答案 → JSONL 文件
  2. evaluate_sqa_response.sh
    • 读取预测结果
    • 与标准答案对比
    • 输出准确率 → JSON 文件

6.2 量化方法流程

[预训练 LVLM] → [加载模型 4bit] → [校准阶段] → [搜索最优量化范围] → [推理评估]
                                              ↓
                                    [激活量化] ← [基于熵的搜索]
                                              ↓
                                       [W4A4 量化]

7. 安装与使用

7.1 安装依赖

# 1. 安装基础包
pip install -e .
