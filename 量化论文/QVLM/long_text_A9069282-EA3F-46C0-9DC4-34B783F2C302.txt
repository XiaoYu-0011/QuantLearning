 一、核心创新点

 ### 1. 跨层依赖挖掘 (Cross-layer Dependency Mining, CDM)

 问题： 传统 PTQ 方法逐层搜索量化参数，忽略了层与层之间的离散化误差依赖关系。

 创新：
 - 发现激活熵 (Activation Entropy) 与跨层依赖强相关
 - 用熵作为代理指标，将模型划分为多个块 (block)
 - 块内联合搜索量化参数，最小化块输出离散化误差

 核心公式：
 $$D(k_r, k_s) = -\sum_{k=k_r}^{k_s} \sum_{ij} p(x_{q,ij}^{(k)}, x_{q,ij}^{(k+1)}) \log p(x_{q,ij}^{(k+1)} | x_{q,ij}^{(k)})$$

 ────────────────────────────────────────────────────────────────────────────────

 ### 2. 视觉编码器优化 (Visual Encoder Optimization, VEO)

 问题： 视觉编码器显著改变激活分布，影响量化搜索效率。

 创新：
 - 优化视觉编码器以最小化激活熵
 - 解耦跨层依赖，实现更细粒度的搜索空间分解
 - 损失函数包含三项：
     - $L_{reg}$: 自回归损失
     - $L_{ent}$: 激活熵最小化 (Jacobian 加权)
     - $L_{err}$: 视觉编码器 + LVLM 输出误差

 核心公式：
 $$L = L_{reg} + \lambda_1 L_{ent} + \lambda_2 L_{err}$$

 ────────────────────────────────────────────────────────────────────────────────

 ### 3. W4A4 量化 SOTA

 成果：
 - 首个实现 LLaVA W4A4 量化的工作
 - 13B 模型：内存压缩 2.78×，生成速度提升 1.44×
 - ScienceQA 精度无损 (79.79% vs FP 80.86%)

 四、与你研究 (KV Cache 量化) 的关联 ⚡

 ### 4.1 核心启发

 ```
   ┌─────────────────────────────── ──────────────────────────────┐
   │  方向 1: 跨层依赖感知的 KV Cache 量化                       │
   │  - Q-VLM: 层间离散化误差有依赖关系                         │
   │  - 迁移：KV Cache 量化时考虑 layer 间的误差传播             │
   │  - 方法：用激活熵识别敏感层，敏感层用高精度                 │
   └─────────────────────────────── ──────────────────────────────┘

   ┌─────────────────────────────── ──────────────────────────────┐
   │  方向 2: 激活熵作为量化敏感度指标                           │
   │  - Q-VLM: 高熵 = 高敏感度 = 需要联合搜索                    │
   │  - 迁移：KV Cache 中，高熵 token 用高精度量化               │
   │  - 类似：VIST 的频率掩码，但用熵代替频率                    │
   └─────────────────────────────── ──────────────────────────────┘

   ┌─────────────────────────────── ──────────────────────────────┐
   │  方向 3:
   │  方向 3: 视觉编码器优化 → KV Cache 投影层优化               │
   │  - Q-VLM: 优化视觉编码器降低激活熵                         │
   │  - 迁移：优化 KV Cache 的投影层 (Q/K/V)降低量化误差         │
   │  - 方法：微调投影层权重，最小化量化后输出误差               │
   └─────────────────────────────── ──────────────────────────────┘

   ┌─────────────────────────────── ──────────────────────────────┐
   │  方向 4: 块级联合搜索 → KV Cache 分层量化                   │
   │  - Q-VLM: 块内联合搜索量化参数                             │
   │  - 迁移：KV Cache 按 layer block 分组，块内统一量化策略     │
   │  - 优势：减少搜索空间，保持跨层一致性                       │
   └─────────────────────────────── ──────────────────────────────┘
 ```
